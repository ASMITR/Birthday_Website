# 🎂 Birthday 3.0 Light Edition

A sweet, heartfelt birthday surprise website built using **Next.js + Tailwind CSS + Framer Motion** — perfect for surprising someone special on their birthday 🎂💖

---

## ✨ Features

- 🎉 Animated birthday greeting with smooth entrance effects
- 💌 Personalized message display
- 🎨 Beautiful, responsive UI with modern design
- ⚡ Fast performance powered by Next.js
- 📱 Fully mobile-friendly layout

> ⚠️ This is a **free/light version** — some features like background music, balloon screen, and final surprise screen are not included.

---

## 🛠 Tech Stack

| Technology | Purpose |
|---|---|
| ⚛️ Next.js | React framework for fast UI |
| 🎨 Tailwind CSS | Modern responsive styling |
| 🎞️ Framer Motion | Smooth entrance & fade animations |

## 🖥 Local Setup

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

# Navigate into the folder
cd YOUR_REPO_NAME

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

---

## 🚀 Deploy on Vercel

1. Push your code to GitHub
2. Go to [vercel.com](https://vercel.com) → **New Project**
3. Import your GitHub repo
4. Click **Deploy** — live in seconds! ✅

---

## 🎨 Customization

To personalize the website, edit the following:

- **Name & message** – Update in `app/page.js`
- **Photos** – Replace images in the `public/` folder
- **Colors & fonts** – Modify in `tailwind.config.js` or `globals.css`
